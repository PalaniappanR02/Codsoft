# AGENTS.md

## Project Context

This is a React + Vite ecommerce application. Keep changes focused on the user's request and preserve existing project conventions.

## Key Files

- `src/` — frontend application source
- `src/api/` — API client
- `src/libs/` — context providers and utilities
- `src/pages/` — page components
- `src/components/` — shared UI components
- `vite.config.js` — Vite configuration

## Working Notes

- Use `npm run dev` to start the local development server
- Run `npm run lint` before finishing code changes
- Reuse existing context providers and API client patterns
